How the Cleverpath pipeline works

  1. 1

    Functional user opens a GitHub issue

  2. 2

    Cleverpath receives the webhook and creates a branch

  3. 3

    AI worker generates CDS, RAP, services, annotations, and tests

  4. 4

    Tests run in a sandbox client

  5. 5

    Senior developer reviews the PR

  6. 6

    On merge, code is pulled into SAP via abapGit and transport is created

  7. 7

    Every action is logged and attributable
